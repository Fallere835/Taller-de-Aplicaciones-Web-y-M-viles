Conceptos que veremos

En esta demostración, aplicaremos estilos a elementos usando selectores de tipo, clase e ID.

Este es un `div` especial para demostrar el Modelo de Caja. Fíjate en el espacio interior (padding), el borde y el espacio exterior (margin) que lo separa de otros elementos.

El Modelo de Caja es fundamental para entender cómo se posicionan los elementos en la página.
